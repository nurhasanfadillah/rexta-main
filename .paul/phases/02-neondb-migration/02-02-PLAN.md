    **FILE 1: `api/products.ts`**
    Import: `db` dari `'../lib/db'`, `requireSession` dari `'../lib/auth-middleware'`
    Semua methods (GET/POST/PUT/DELETE) memerlukan auth.

    GET (pagination + filters):
    - Parse query params: `page` (default 1), `limit` (default 20), `search`, `categoryId`, `onlyFavorites`
    - offset = (page - 1) * limit
    - Build dynamic WHERE — gunakan pola ini:
      ```
      const conditions: string[] = [];
      const params: any[] = [];
      let idx = 1;

      if (search) { params.push(`%${search}%`); conditions.push(`name ILIKE $${idx++}`); }
      if (categoryId && categoryId !== 'SEMUA') { params.push(categoryId); conditions.push(`category_id = $${idx++}`); }
      if (onlyFavorites === 'true') { conditions.push(`is_favorite = true`); } // boolean literal, no param

      const where = conditions.length ? `WHERE ${conditions.join(' AND ')}` : '';
      ```
    - Jalankan 2 queries secara paralel (Promise.all):
      1. `SELECT COUNT(*) as total FROM products ${where}` → pakai `params`
      2. `SELECT * FROM products ${where} ORDER BY name ASC LIMIT $${idx} OFFSET $${idx+1}` → pakai `[...params, limit, offset]`
    - Response: `{data: rows.map(mapProductFromDB), count: parseInt(countRow.total)}`

    POST (body fields dalam snake_case: id, name, category_id, price_cmt, hpp, stock, is_favorite):
    - INSERT INTO products (id, name, category_id, price_cmt, hpp, stock, is_favorite, updated_at) VALUES ($1...$7, NOW()) RETURNING *
    - Response: mapProductFromDB(result.rows[0])

    PUT (query param `?id=xxx`, body fields sama dengan POST minus id):
    - UPDATE products SET name=$1, category_id=$2, price_cmt=$3, hpp=$4, stock=$5, is_favorite=$6, updated_at=NOW() WHERE id=$7 RETURNING *
    - Response: mapProductFromDB(result.rows[0])

    DELETE (query param `?id=xxx`):
    - DELETE FROM products WHERE id = $1
    - Response: `{success: true}`

    Response:
    ```json
    {
      "data": [{"id": "...", "name": "...", "stock": 0, "categoryId": "..."}],
      "categories": [{"id": "...", "name": "..."}],

    }
    ```
