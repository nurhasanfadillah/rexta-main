  <name>Task 1: Buat Materials CRUD API</name>
  <files>api/materials.ts</files>
  <action>
    Buat file `api/materials.ts` dengan pola identik `api/products.ts`:

    1. Import: `import { db } from '../lib/db'` dan `import { requireSession } from '../lib/auth-middleware'`

    2. Buat `mapMaterialFromDB(row)`:
       ```
       { id, name, unit, price: Number(row.price)||0, stock: Number(row.stock)||0, updatedAt: row.updated_at }
       ```

    3. `export default async function handler(req, res)`:
       - Jalankan `await requireSession(req)` di awal, catch → return 401

    4. GET handler:
       - Parse `page` (default 1), `limit` (default 20), `search` dari `req.query`
       - Dynamic WHERE: jika `search` ada, tambah `name ILIKE $N` ke conditions[]
       - Parallel: `db.query(COUNT)` + `db.query(SELECT * ... ORDER BY name ASC LIMIT $N OFFSET $N)`
       - Return `{ data: rows.map(mapMaterialFromDB), count: parseInt(total) }`

    5. POST handler:
       - Destructure body: `{ id, name, unit, price, stock }` — snake_case, konsisten DB
       - INSERT dengan `updated_at = NOW()`
       - Return `mapMaterialFromDB(result.rows[0])`

    6. PUT handler:
       - `id` dari `req.query.id`
       - Destructure body: `{ name, unit, price, stock }`
       - UPDATE SET name, unit, price, stock, updated_at=NOW() WHERE id=$N
       - `rowCount === 0` → return 404
       - Return `mapMaterialFromDB(result.rows[0])`

    7. DELETE handler:
       - `id` dari `req.query.id`
       - DELETE FROM materials WHERE id=$1
       - Return `{ success: true }`

    8. Fallback: return 405 Method Not Allowed

    Error handling: setiap method dalam try/catch terpisah.
    Log prefix: `[materials]` (contoh: `console.error('[materials] GET error:', error)`)

    Avoid:
    - Jangan gunakan `sql` template literal untuk query dinamis (gunakan `db.query()` saja)
    - Jangan skip `rowCount` check di PUT
    - Jangan buat endpoint public (semua materials protected)
  </action>
