<objective>
## Goal
Fix false-negative login: ganti `data?.session` → `data` di LoginView.tsx sehingga login dengan kredensial benar langsung menampilkan notif sukses dan masuk dashboard — tanpa perlu refresh.

## Purpose
Saat ini login dengan kredensial benar menampilkan error "Login gagal. Periksa email dan password." karena `authClient.signIn.email()` mengembalikan `{ data: { token, user } }` — bukan `{ data: { session } }`. Session cookie sudah di-set server-side, tapi client-side check gagal. Satu baris fix ini memperbaiki UX login sepenuhnya.

## Output
`components/LoginView.tsx` diperbarui (1 baris).
</objective>
